Git Code Manager Tool / Git代码管理工具
========================================

Files Created / 创建的文件:
- git-tool.bat         (Main program / 主程序)
- create-shortcut.bat  (Create desktop shortcut / 创建桌面快捷方式)

Quick Start / 快速开始:
1. Double-click "create-shortcut.bat" / 双击 "create-shortcut.bat"
2. Use desktop shortcut "Git Code Manager" / 使用桌面快捷方式 "Git Code Manager"

Menu Options / 菜单选项:
1. Check Status          / 查看状态
2. Setup Remote Repo     / 设置远程仓库
3. Commit and Upload     / 提交并上传代码
4. Download Updates      / 下载更新
5. Open Project Folder   / 打开项目文件夹
6. Show Help            / 显示帮助
0. Exit                 / 退出

First Time Setup / 首次设置:
1. Select option 2 (Setup Remote Repository) / 选择选项2 (设置远程仓库)
2. Choose platform: GitHub or Gitee / 选择平台: GitHub 或 Gitee
3. Create repository on chosen platform / 在选择的平台创建仓库
4. Enter repository URL / 输入仓库地址
5. Use option 3 to upload code / 使用选项3上传代码

Repository URL Examples / 仓库地址示例:
GitHub: https://github.com/username/feishu-dashboard.git
Gitee:  https://gitee.com/username/feishu-dashboard.git

Notes / 注意事项:
- All interface is in English to avoid encoding issues
- 所有界面都是英文以避免编码问题
- Functions are self-explanatory and easy to use
- 功能一目了然，易于使用
- Built-in help provides detailed guidance
- 内置帮助提供详细指导

Support / 技术支持:
If you encounter issues, check:
如果遇到问题，请检查:
1. Network connection / 网络连接
2. Repository URL format / 仓库地址格式
3. Repository permissions / 仓库权限
